# SpamDestroyer
Naive Bayesian Spam Filter written in Java using JavaFX as a class assignment for Software Systems Development and Integration. JavaFX UI was implemented programatically.

## Train.java
Generates bag of words models, and determines the 'spamliness' of a word given by a bayesian probability.

## Test.java
Tests the trained model against a set of spam and 'ham' emails.
